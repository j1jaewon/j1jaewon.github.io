# BRIDGE Project

**Full name**: Barrier Reduction & International Deployment Guidance Engine

## Problem Statement
Korean SMEs exporting cosmetics to the EU face:
- High risk of customs rejection due to ingredient non-compliance
- High consulting costs for regulatory pre-screening
- Lack of dedicated regulatory staff

## Solution
AI-powered web tool that pre-verifies cosmetic ingredient compliance
against EU regulations **before** the product is exported.

## Target Users
SME practitioners in cosmetics export with limited regulatory expertise

## Key Features (from spec)
- Input: product ingredient list
- Processing: AI checks against EU Cosmetics Regulation (EC 1223/2009), REACH, banned/restricted substance lists
- Output: compliance assessment with flagged ingredients and risk level

## Technical Approach
- Web page implementation
- GitHub integration for version control
- AI-assisted ingredient matching against regulatory database

## Status
- Spec drafted and web page prototype implemented (Claude Code session)
- GitHub integration explored

## Broader Significance
This project is a prototype for the [[future-plans]] vision of automating
regulatory compliance checks — scaling the [[ai-workflow-tbt-report]] approach
into a self-service tool for exporters.
